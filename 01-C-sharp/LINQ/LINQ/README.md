# LINQ Learning Journey


This repository has been reorganized to isolate each LINQ topic into its own independent project. This makes it easier to navigate, experiment, and run individual examples.

### Repository Structure
- **Topics/**: Contains all LINQ operators and concepts.
    - Each sub-folder is a standalone .NET Console Application.
    - You can run any specific topic by navigating to its folder and using `dotnet run`.
